# Plan 10-03: Frontend Dashboard Implementation

## Objective
Implement comprehensive dashboard UI components to display operational data, marketing analytics, customer insights, product metrics, and integration health using data from new API endpoints.

## Context
Current dashboard shows: KPI cards, revenue trend, recent sales, empty marketing events, placeholder operational data. Need to populate all sections with real data.

## Tasks

### Task 1: Create Dashboard Data Hook
**Files**: `admin-ui/src/hooks/useDashboard.ts` (new)

Create custom hook to fetch all dashboard data in parallel:
```typescript
export function useDashboard() {
  const [data, setData] = useState({
    operational: null,
    marketing: null,
    customers: null,
    products: null,
    integrations: null
  });
  
  const refresh = async () => {
    const [operational, marketing, customers, products, integrations] = 
      await Promise.all([
        api.get('/dashboard/operational'),
        api.get('/dashboard/marketing'),
        api.get('/dashboard/customers'),
        api.get('/dashboard/products'),
        api.get('/dashboard/integrations')
      ]);
    setData({ operational, marketing, customers, products, integrations });
  };
  
  return { ...data, refresh };
}
```
