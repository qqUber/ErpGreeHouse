## Overview
ErpGreeHouse is an omnichannel CRM + Loyalty system for coffee shops, designed to replace expensive third-party solutions with an open-source alternative. The system will unify customer loyalty management and targeted communications across Telegram, VK, and mobile app channels.

## Core Requirements

### Loyalty Program Management
- **✓ LOY-001**: User can check loyalty points balance via any supported channel — v1.0
- **✓ LOY-002**: Loyalty points are automatically accrued based on purchase amount — v1.0
- **✓ LOY-003**: Loyalty points can be redeemed for purchases — v1.0
- **✓ LOY-004**: Loyalty program data is synchronized across all channels — v1.0

### Messaging System
- **✓ MSG-001**: Send targeted promotional messages to customer segments — v1.0
- **✓ MSG-002**: Implement trigger-based messages (birthday, inactivity, welcome) — v1.0
- **✓ MSG-003**: Support for text and media messages (images, videos, documents) — v1.0
- **✓ MSG-004**: Rate limiting to prevent channel bans — v1.0
- **✓ MSG-005**: Track message delivery and open rates — v1.0

### Compliance & Data Protection
- **✓ COMP-001**: Explicit consent collection for personal data processing — v1.0
- **✓ COMP-002**: Consent records include timestamp and document version — v1.0
- **✓ COMP-003**: User can revoke consent and delete profile in 1 click — v1.0
- **✓ COMP-004**: All data stored within Russian Federation borders — v1.0

### ERP Integration
- **✓ ERP-001**: Sync customer data with ERPNext — v1.0
- **✓ ERP-002**: Import purchase data from ERP for loyalty point calculation — v1.0
- **✓ ERP-003**: Export loyalty program data to ERP — v1.0

### Admin Dashboard
- **✓ ADMIN-001**: View and manage customer profiles — v1.0
- **✓ ADMIN-002**: Create and manage promotional campaigns — v1.0
- **✓ ADMIN-003**: View messaging analytics and reports — v1.0
- **✓ ADMIN-004**: Configure loyalty program settings — v1.0
