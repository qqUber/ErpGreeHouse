# Результаты нагрузочного тестирования (MVP)

## Контекст

Текущая реализация MVP ориентирована на демонстрацию UI/UX и корректность бизнес-потоков.
Для высоконагруженной эксплуатации (до 1 500 000 одновременных пользователей) требуется переход на промышленную архитектуру (см. `ha_scaling.md`).

Тем не менее, выполнена базовая нагрузочная проверка API-эндпоинтов, чтобы:
- устранить ошибку «сервис недоступен» при первом открытии;
- зафиксировать текущую производительность на локальном окружении;
- выделить узкие места.

## Инструмент

Скрипт: `middleware/tools/load_test.py`

Параметры:
- `LOAD_BASE_URL` — базовый URL
- `LOAD_ENDPOINT` — путь без ведущего `/` (особенность Git Bash на Windows)
- `LOAD_USERS` — конкурентность
- `LOAD_SECONDS` — длительность
- `ADMIN_SECRET` — требуется для защищённых эндпоинтов

## Запуски

### 1) Public статус (без авторизации)

Команда:

```bash
LOAD_BASE_URL='http://127.0.0.1:8000/' LOAD_ENDPOINT='api/v1/public/status' LOAD_USERS=50 LOAD_SECONDS=5 py tools/load_test.py
```

Результат:
- rps ≈ 207.8
- latency (ms): avg ≈ 247.4, p50 ≈ 174.9, p95 ≈ 684.9, p99 ≈ 1093.3

### 2) Dashboard (защищено x-admin-secret)

Команда:

```bash
LOAD_BASE_URL='http://127.0.0.1:8000/' LOAD_ENDPOINT='api/v1/dashboard' LOAD_USERS=50 LOAD_SECONDS=5 ADMIN_SECRET='change-me' py tools/load_test.py
```

Результат:
- rps ≈ 211.0
- latency (ms): avg ≈ 241.0, p50 ≈ 156.8, p95 ≈ 700.5, p99 ≈ 1030.8

## Интерпретация

- Метрики получены на локальном окружении Windows с ограничениями IO и без оптимизированной инфраструктуры.
- Для целей «enterprise-scale» эти числа не являются целевыми; они служат baseline.

## Дальнейшие шаги для целевых нагрузок

- Перенести CRM-хранилище с SQLite на PostgreSQL (пул соединений, индексы, реплики).
- Вынести статику UI на CDN/Reverse Proxy, отделив её от API.
- Сделать API строго stateless, добавить горизонтальное масштабирование (несколько реплик).
- Добавить кэширование Redis (уже внедрено для /dashboard и /customers).
- Ввести rate limiting и backpressure, а также SLO/SLA мониторинг.
