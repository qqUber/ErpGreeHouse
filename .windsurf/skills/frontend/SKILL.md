## 1. Modern Frontend
* **React + TypeScript:** Strict Mode, кастомные хуки, мемоизация.
* **Build Tools:** Vite для быстрой сборки и HMR.
* **State Management:** Управление сложными состояниями (Zustand / Redux Toolkit / React Query) для кэширования ответов API.
* **Styling:** Tailwind CSS, создание переиспользуемых UI-компонентов.

## 2. Domain Knowledge & BPMN
* **Business Process Modeling:** Чтение и проектирование схем BPMN 2.0 для синхронизации разработки и бизнеса.
* **ERP Logic:** Понимание процессов инвентаризации, лояльности и управления цепочками поставок.

## 3. UX & Security
* **Zero-Trust UI:** Рендеринг компонентов строго на основе прав доступа (RBAC) из JWT.
* **Real-time Dashboards:** Интеграция WebSockets или Server-Sent Events (SSE) для обновления аналитики в реальном времени.

## 4. AI-Driven UI Testing
* Генерация визуальных тестов и автоматическое обнаружение аномалий верстки.
