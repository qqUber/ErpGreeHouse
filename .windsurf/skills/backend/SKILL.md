## 1. Core Backend (FastAPI)
* **Python:** Глубокое понимание асинхронности, управление зависимостями (uv/Poetry).
* **FastAPI Ecosystem:** Pydantic validation, Dependency Injection, Middleware.
* **Data Layer:** SQLite/PostgreSQL, SQLAlchemy, оптимизация запросов и миграции (Alembic).
* **Caching & Queue:** Redis для rate limiting и брокеров сообщений.

## 2. ERPNext & Integrations
* **ERPNext API:** Разработка отказоустойчивых коннекторов, REST/RPC вызовы.
* **Telegram Bots:** Aiogram 3.x для оперативного уведомления и управления процессами.
* **Data Sync:** Обеспечение консистентности данных между ERP и локальной базой.

## 3. DevOps & Infrastructure
* **Containerization:** Docker, многоэтапные сборки, оптимизация размеров образов.
* **CI/CD Pipeline Engineering:** GitHub Actions (linting, testing, multi-environment deployments).

## 4. AI-Assisted Development
* **Local AI Integration:** Настройка и использование MCP серверов (filesystem, fetch, sequential thinking) для локальных сред разработки.
